Smart Attendance System

Automated attendance using face recognition

Project Overview

This system is designed to automate attendance management, eliminating manual record-keeping and preventing proxy attendance. It uses face recognition technology to accurately identify students and update attendance records in real-time. This project was developed as a part of our Bachelor of Engineering program.
